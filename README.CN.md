# PigSQL
豚豚SQL是一个简化 .net 开发接口的 SQL Server 工具集类库。

## ***目录和文件描述***

### Setup

执行码目录，如果你不想看到源程序，你可以直接使用这个目录中的文件。

##### Setup\Core3.1\PigSQLLib.dll 和 

豚豚SQL类库文件，将这个文件引用到你的 .net 工程就可以使用。

执行码编译使用 .Net Core 3.1，可以运行在 Windows、Linux 和 MacOS 平台。

#### Src\Core3.1\PigSQLLib.1.0.0.nupkg

PigSQLLib 的 NuGet 包。

##### Setup\DotNet4.5\PigSQLLib.dll

豚豚SQL类库文件，将这个文件引用到你的 .net 工程就可以使用。

执行码编译使用 Net framework 4.5，只能运行在 Windows 平台。
